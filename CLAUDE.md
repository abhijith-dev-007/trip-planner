# Project Instructions

## Project
Build a production-ready web application using:
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB
- Auth: JWT
- Styling: Tailwind CSS

## Rules
- Do not rewrite the whole project unless asked.
- First inspect existing files before editing.
- Make small commits/changes per task.
- Explain only important changes.
- Prefer reusable components and clean folder structure.
- Always update related API, DB, and UI files together.
- Do not add unnecessary libraries.
- Ask before major architecture changes.

## Commands
Frontend:
- npm install
- npm run dev
- npm run build

Backend:
- npm install
- npm run dev

## Coding Style
- Use functional React components.
- Use async/await.
- Use proper error handling.
- Keep controllers, services, routes, and models separate.
- Validate API inputs.
